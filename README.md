
# 🏏 IPLytics

AI-Powered IPL Match Prediction & Sports Intelligence Platform

IPLytics is a full-stack sports analytics platform that predicts IPL match outcomes using Machine Learning and provides interactive dashboards for deeper team insights.

---

## 🚀 Features

* Match Winner Prediction
* Win Probability Estimation
* Team Momentum Analytics
* Interactive Dashboard
* Machine Learning Powered Predictions
* Modern Responsive UI
